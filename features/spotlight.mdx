Ori's Spotlight mode lets you summon the AI from **any application** with a single keystroke. No window switching, no context breaking.

## How to use it

1. Press **`⌥ Space`** (Option + Space) from anywhere
2. Ori appears as a floating overlay
3. Type your question or command
4. Get your answer
5. Press **`⌥ Space`** again to dismiss (or click outside)

## What makes it powerful

Spotlight is designed for quick interactions — the AI equivalent of Spotlight Search:

| Use case | Example |
|:---------|:--------|
| Quick question | "What's the CSS property for text shadow?" |
| Quick calculation | "Convert 72°F to Celsius" |
| Screen context | "What's the error on my screen?" (takes a screenshot) |
| Quick command | "Create a new file called notes.md in ~/Desktop" |
| Memory lookup | "What was that API endpoint I used last week?" |

## Always-on

Ori runs in your **system tray** (menu bar on macOS), so the `⌥ Space` hotkey is always available — even when the main Ori window is closed. As long as Ori is running in the tray, Spotlight works.

<Tip>
  Spotlight has full access to all of Ori's capabilities — memory, tools, computer use, connectors. It's not a limited mode — it's the same Ori, just summoned differently.
</Tip>
