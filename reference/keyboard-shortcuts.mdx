## Navigation

| Shortcut | Action |
|:---------|:-------|
| `⌘ N` | New conversation |
| `⌘ K` | Search threads (Command Palette) |
| `⌘ ,` | Open Settings |
| `` Ctrl+` `` | Toggle terminal panel |

## Chat

| Shortcut | Action |
|:---------|:-------|
| `Enter` | Send message |
| `Shift + Enter` | New line in the prompt |
| `Escape` | Stop streaming / close overlays |
| `⌘ S` | Save file in preview panel (edit mode) |

## Preview panel

| Shortcut | Action |
|:---------|:-------|
| `⌘ S` | Save the file being edited |
| Click file in sidebar | Opens file in preview panel |
| Click same file again | Closes preview panel |

<Tip>
  The `⌘ K` shortcut is the fastest way to find and switch between conversations. Threads are grouped by time — today, yesterday, this week, and older.
</Tip>
